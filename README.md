#
## 📦 Instalación

```bash
# Instalar dependencias
npm install

# O con yarn
yarn install
```

## 🛠️ Comandos Disponibles

```bash
# Desarrollo (http://localhost:3000)
npm run dev

# Build de producción
npm run build

# Iniciar servidor de producción
npm run start

# Ejecutar tests
npm run test

# Ejecutar tests en modo watch
npm run test:watch

# Linting
npm run lint

# Formatear código
npm run format
```
